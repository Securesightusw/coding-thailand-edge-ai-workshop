# Pull Request

## สรุปการเปลี่ยนแปลง

_เขียนสั้นๆ ว่า PR นี้ทำอะไร_

## Type of Change

- [ ] 🚀 feat — เพิ่ม feature ใหม่
- [ ] 🐛 fix — แก้ bug
- [ ] 📊 data — เพิ่ม/แก้ dataset
- [ ] 🏋️ train — train model ใหม่
- [ ] 📝 docs — แก้เอกสาร
- [ ] 🧪 test — เพิ่ม/แก้ test cases
- [ ] ♻️ refactor — restructure code

## Why?

_ทำไมต้องเปลี่ยน — บอกเหตุผล ไม่ใช่แค่ "ทำอะไร"_

## How Tested?

- [ ] Run บน UNO Q จริง
- [ ] Test cases ผ่าน
- [ ] Accuracy เพิ่ม/คงเดิม
- [ ] Demo ทำงานได้

## Screenshots/Recording

_ถ้ามี attach ไว้_

## Checklist

- [ ] Commit messages ตาม format `<type>: <what> เพราะ <why>`
- [ ] ไม่มี secret/large file (เช็ค .gitignore)
- [ ] ทีมในที่เห็น PR นี้แล้ว
